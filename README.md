DjangoTodo
A simple Todoapp built with Django where users can create, edit and delete their todo.

Features
Create todo
edit todo
update taot
mark completed todo
delete todo

Tech Stack
Django
SQLite
HTML, CSS
bootstrap

Setup Instructions
git clone https://https://github.com/yjjwal/djangotodo.git
cd todos
python -m venv env
env\Scripts\activate 
#Active virtual environment in windows
pip install -r requirements.txt
python manage.py runserver
